Commencer

Résoudre un 4x4x4 à l'aveugle est probablement plus facile que vous ne le pensez, si vous savez déjà résoudre un 3x3x3 à l'aveugle. Vous devrez mémoriser environ 25 paires de lettres, ce que vous pouvez faire efficacement en utilisant des pièces romaines. Il n'y a pas beaucoup d'algorithmes à apprendre, amusez-vous bien et bonne chance !


Pré-requis :

  • M2 pour les arêtes
  • OP pour les coins
  • Temps et dévouement

    • Algorithmes d’échange

  • Centre : U2
  • Aile : r2

    • Règles d’échange

  • Dans la plupart des cas, utilisez : setup + swap + undo setup
  • Pour les cas spéciaux, utilisez uniquement l’algorithme indiqué
  • Centres : Si B ou D est en deuxième dans une paire de lettres, échangez-le avec l’autre
  • Ailes : Si I ou S est en deuxième dans une paire de lettres, échangez-le avec l’autre

    • Algorithmes de parité

  • Centre : U2
  • Aile : r' U2 r U2 r' U2 x r U2 r U2 r U2 r2 U2 x' r' U2
  • Cornet : U2 R U R' U' (r2 U2 r2 Uw2 r2 Uw2) U' R U' R' U2

    • Règles de parité

  • Si le nombre de cibles (lettres) est impair pour une étape, faites l’algorithme de parité après l’avoir résolue
  • La parité est indépendante pour chaque étape (centres, ailes, coins)
  • Ordre d’exécution : Centres → Ailes → Coins (important pour éviter de casser les pièces)

    • Comment savoir quand j’ai fini de mémoriser ?

    Vous pouvez utiliser cette formule simple qui vous indiquera combien vous devez mémoriser : Pièces - 1 - Pièces résolues + Nouveaux cycles

    Par exemple : il y a 24 ailes - 1 = 23, j’ai 2 pièces résolues, 23 - 2 = 21 mais j’ai 3 nouveaux cycles : 21 + 3 = 24. Si j’ai mémorisé seulement 19 pièces, il m’en manque 5.

    Vous pouvez appliquer cela aussi aux centres.

    Mouvements de setup pour centres

  • A : tampon
  • B : (spécial) Rw' F' U r U' l' U r' U' l F Rw U2
  • C : aucun
  • D : (spécial) Lw F' r U l' U' r' U l U' F Lw' U2
  • E : r u r'
  • F : y' u r' u' r
  • G : r' d r
  • H : y' r d2 r'
  • I : r u2 r'
  • J : y' r' u r
  • K : d' r' d r
  • L : y' r d r'
  • M : r u' r'
  • N : y' r' u2 r
  • O : r' d' r
  • P : y' d' r d r'
  • Q : u r u' r'
  • R : y' r' u' r
  • S : r' d2 r
  • T : y' r d' r'
  • U : (spécial) D [échanger avec V] D'
  • V : (spécial) Lw' U' r2 U' l U r2 U' l' U2 Lw U2
  • W : (spécial) D' [échanger avec V] D
  • X : (spécial) D2 [échanger avec V] D2

    • Mouvements de setup pour ailes

  • A : aucun
  • B : R U R' U'
  • C : (spécial) l' [échanger avec Q] l
  • D : L' U' L U
  • E : B L' B'
  • F : B L2 B'
  • G : B L B'
  • H : L B L' B'
  • I : (spécial) D r U R2 U' r' U R2 U' D' r2
  • J : U R U'
  • K : (spécial) l2 [échanger avec Q] l2
  • L : U' L' U
  • M : B' R B
  • N : R' B' R B
  • O : B' R' B
  • P : B' R2 B
  • Q : (spécial) U B' R U' B (r2) B' U R' B U'
  • R : U' L U
  • S : (spécial) r2 D U R2 U' r U R2 U' r' D'
  • T : U R' U'
  • U : tampon
  • V : U R2 U'
  • W : (spécial) l [échanger avec Q] l'
  • X : U' L2 U
