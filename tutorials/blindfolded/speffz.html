What is the Speffz Lettering Scheme?

The Speffz Lettering Scheme (short for Sticker Position Encoding for Fewest-move and Freestyle Zones) is a standardized system for assigning letters to the stickers on a Rubik’s Cube. It helps cubers:

  • Memorize pieces faster using letter pairs
  • Use the same language across tutorials
  • Practice with consistency across sticker-based methods

    • How it works?

    The cube's stickers (not the pieces!) are labeled with letters from A to X, following a fixed order for each face.

  • Each face has 4 stickers (excluding centers), and each face is assigned a set of letters.
  • Speffz goes clockwise, starting from the top-left sticker on each face.
