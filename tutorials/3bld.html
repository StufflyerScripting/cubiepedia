What is 3BLD?

3BLD (3×3 Blindfolded) is a WCA event where you inspect the cube, then solve it fully blindfolded using memory and special solving techniques like Old Pochmann or M2/3-style.

Requirements

  • Basic 3×3 knowledge
  • Comfortable with solving last layer
  • Willingness to memorize fast

Recommended Methods

  • Old Pochmann – for beginners
  • M2 for edges + 3-style for corners – for intermediate/advanced

Memo System

Letter pairs are commonly used. The cube is broken down into sticker targets, then converted to letter sequences that are memorized as words.

Example: UB → A, UL → B, etc.

Execution

  1. Blindfold on
  2. Edges first (M2/OP)
  3. Then corners (OP/3-style)

Tips

  • Practice memo with flashcards
  • Use a fixed buffer (like UFR for corners)
  • Review solves with scrambles + memo log
